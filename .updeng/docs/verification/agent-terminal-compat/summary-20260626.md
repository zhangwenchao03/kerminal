## Native Default Behavior

- Agent terminal default UI now renders only the real `XtermPane` plus minimal shell chrome: back button, title/command, target binding and size warning.
- The removed default controls include the prompt composer, paste/send/queue buttons, prompt queue count, explicit waiting status, prompt history, target output, selection, command block and branch helper buttons.
- Agent terminal env normalization converts missing or `dumb` `TERM` to `xterm-256color` and sets `COLORTERM=truecolor` when absent. Explicit non-dumb terminal values are preserved.
- Image paste is not handled through an external Agent prompt box. It remains a terminal-layer compatibility item: validate clipboard image/file drop/path paste and terminal image protocols without replacing the Agent TUI.
